# KnProxy 某个诡异的网页代理

KnProxy是一个轻量级的，基于PHP的网页代理。它可以通过 cURL 或者 PHP Sockets 来将
HTTP/HTTPS 的内容进行重新传导。KnProxy的设计目标是在同类基于改写的网页代理中，提供相对高的
还原度并保持简洁的部署。KnProxy 的最初目标是用于临时快速架设一个简单粗暴的穿越 G-F-W 的
网络代理。

KnProxy一般不需要任何更改就可使用，不过你可以在设置中对其进行微调。
